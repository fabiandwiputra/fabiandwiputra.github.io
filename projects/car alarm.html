Simple Car Alarm System

  • Tech Stack: Arduino, Sensors, Buzzer, Wireless Modules, C++
  • Github URL: project link

The Simple Car Alarm System is designed to enhance vehicle security by detecting unauthorized access or tampering. It uses sensors to monitor doors, windows, and motion inside the vehicle, triggering an alarm to alert the owner and deter theft.

This system is easy to install and can be integrated with existing vehicle electronics. It provides real-time alerts and helps protect the vehicle from break-ins and vandalism.

Key Components:

  • Door and Window Sensors: Detect if doors or windows are opened without authorization.
  • Motion Sensor: Detects movement inside the vehicle cabin.
  • Microcontroller (e.g., Arduino): Processes signals from sensors and controls the alarm.
  • Buzzer or Siren: Emits loud sound to alert the owner and scare off intruders.
